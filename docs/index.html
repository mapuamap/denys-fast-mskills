Два способа планировать

Лёгкий

Контекст в infra.md + deploy.md, управляется /m_infr* и /m_deploy*. Быстро, для устоявшихся проектов.

Формальный

Конвейер артефактов m_plan с оракулом проверки (09_verification.md). Для крупных или рискованных изменений.

Установка

Добавь маркетплейс

claude plugin marketplace add mapuamap/denys-fast-mskills

Установи плагин

claude plugin install denys-fast-mskills@denys-fast-mskills

Перезапусти Claude Code, затем…

$ /m_setup · /m_plan <task>

Внутри Claude Code можно также использовать команды /plugin marketplace add и /plugin install.
