Плагин для Claude Code

Дисциплинированный m_*-набор для Claude

План → выполнение → проверка. Код, инфра, деплой и браузерные тесты — из одного плагина.

Вставь промпт в Claude Code — он всё поставит сам:

Установи мне плагин Claude Code denys-fast-mskills: выполни `claude plugin marketplace add mapuamap/denys-fast-mskills`, затем `claude plugin install denys-fast-mskills@denys-fast-mskills`, затем `claude plugin list` для проверки, и скажи, когда перезапустить Claude Code.
Или поставить вручную командами
/plugin marketplace add mapuamap/denys-fast-mskills /plugin install denys-fast-mskills@denys-fast-mskills

Затем перезапусти Claude Code и запусти /m_plan.

Customize → Plugins → Add marketplace — вставь репозиторий и нажми Install:

mapuamap/denys-fast-mskills

Команды и агенты — только в Claude Code; на Desktop/web доступны скилы.

Нет «Add marketplace»? Загрузи скилы как .zip (Customize → Skills → Upload):
m_plan m_plan_implement m_code_init_project m_code_refactor m_code_rules_audit
